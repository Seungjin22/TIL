이것은 block 요소, 기본적으로 너비의 100% 차지

너비가 정해지면 나머지를 margin으로!
block의 오른쪽 마진을 auto로 주면 왼쪽으로 붙고
block의 왼쪽 마진을 auto로 주면 오른쪽으로 붙는다.
block의 왼쪽/오른쪽 마진을 auto로 주면 가운데 정렬
중앙정렬은 그냥 margin auto로 하면 됩니다.

p태그는 원래 block level 요소지요?

이제는 inline-block 요소가 되어 붙을 거에여.

이건 display none 속성입니다. 요소가 보이지도 않고 영역도 차지하지 않아요.
이건 해당 요소를 안보이게 하지요. display none과 다른건 이 친구는 공간을 차지합니다!(기둥 뒤 공간 있어요!)
